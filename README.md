# Sistema de Información para Microempresa

Proyecto académico desarrollado para mejorar la gestión y control de procesos administrativos en una microempresa de Valledupar.

## Funcionalidades

- Inicio de sesión
- Registro de clientes
- Gestión de inventario
- Registro de ventas
- Reportes

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- GitHub

## Herramientas
- Visual Studio Code
- XAMPP
- Git
