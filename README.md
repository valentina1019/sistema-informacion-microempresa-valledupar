# Sistema de Información para Microempresa – Valledupar

## Descripción
Proyecto académico orientado al diseño e implementación de un sistema de información para mejorar la gestión y control de procesos administrativos en una microempresa de Valledupar.

## Funcionalidades
- Gestión de clientes
- Control de inventarios
- Registro de ventas
- Reportes administrativos
- Inicio de sesión

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- GitHub

## Nivel TRL
TRL 5 – Prototipo funcional validado en entorno relevante.
